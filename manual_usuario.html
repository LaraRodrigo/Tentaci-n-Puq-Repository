Manual de Usuario

Este sitio web va dirigido para los usuarios finales como para los administradores del sitio web

Requisitos del sistema

Para este sitio web no se necesitan equipos potentes ya que la carga de datos no sera muy grande, para esto hicimos que la base de datos se encargue del manejo de imagenes para que este no sea tan pesado para el hosting, dicho esto el sitio web puede ser abierto por un smartphone.

Acceso al sitio web

Este sitio web puede ser encontrado desde la url : (pagina local que se corre desde sus equipos) localhost/index.html los requisitos para iniciar sesion a este sitio web son los siguientes: tener un correo electronico ya sea Gmail o Outlook , tener contraseña.

Para moverse en este sitio web se creo un motor de busqueda con las palabras claves : Chaquetas, Parkas, Polera, Poleras, Jockey, Jockeys, etc. Tambien otra funcionalidad es el apartado de menu desplegable el cual tambien nos da la posibilidad de movernos más rapidos por los diferentes productos que se ofrecen. Tambien existe un carrusel el cual nos deja ver productos de manera rapida dandonos un vistazo más amplio de estos. login / registro aqui se puede tanto si ya tienes una cuenta como si quieres crearte una cuenta para poder estar en un modo "Administrador" y un cliente "Común" el cual sera dado por el tipo de correo que se utilize.

Funcionalidades Principales

Una de las funcionalidades principales de este sitio web es que se tiene un motor de busqueda para dirigirse a las distintas secciones del sitio haciendo que sea más rapido encontrar los distintos productos que se estan vendiendo en esta empresa, dandoles a los usuarios una mejor manera de encontrar productos que les interese. Otra funcionalidad que esta en el prototipo seria el apartado de registro y login el cual tiene la funcionalidad de subir las cuentas creadas a una base de datos la cual manejara los datos de los clientes para que sean reconocidos; el login es una funcionalidad que se creo apartir del registro de datos de nuestros modulos, haciendo capaz de consultar a la pagina si en la base de datos existe una persona con el Usuario llamado por ejemplo "Matias" y si es asi dejame iniciar sesion con este nombre y una contraseña que fue "Hasheada" (hasehar :encriptar la contraseña para mayor seguridad , ya que estos no se pueden "regresar a su estado original") para obtener seguridad en sus datos.
